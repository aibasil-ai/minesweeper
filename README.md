# 踩地雷

這是一個以 TypeScript 撰寫的踩地雷遊戲，支援多難度與自訂棋盤，並包含音效、排行榜、提示與鍵盤操作。

## 操作
- 左鍵開啟格子
- 右鍵插旗
- 行動裝置長按插旗
- 鍵盤：方向鍵移動，空白或 Enter 開啟，F 插旗
- 介面提供音效開關與提示按鈕

## 功能
- 音效：開格、插旗、勝負提示音（可關閉）
- 排行榜：依棋盤設定保存前 5 名最佳時間
- 提示：標示安全格，未開局時可協助安全起手

## 開發與執行
1. 安裝依賴
   - `npm install`
2. 編譯
   - `npm run build`
3. 開啟本地伺服器
   - `npm run dev`
4. 瀏覽器開啟 `http://localhost:8000` 並載入 `index.html`

> 若只想快速體驗，先完成編譯再開啟 `index.html`。
